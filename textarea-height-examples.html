Textarea高度控制方法

方法1: CSS固定高度（推荐用于标题输入）
.single-line { height: 40px; min-height: 40px; max-height: 40px; overflow: hidden; line-height: 20px; }
方法2: 最小高度，可自动增长（推荐用于内容输入）
.auto-grow { min-height: 40px; line-height: 20px; resize: vertical; }
方法3: 使用rows属性
<textarea rows="1"></textarea>
方法4: JavaScript动态调整
const textarea = document.getElementById('dynamic-textarea'); textarea.addEventListener('input', function() { this.style.height = 'auto'; this.style.height = this.scrollHeight + 'px'; });
