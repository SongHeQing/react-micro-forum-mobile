# React MICRO FORUM Moblic
## 开发常用命令
#### 启动项目：
```bash
npm run dev
.\dev.bat # windows bat启动
```
#### 构建项目：
构建生产环境项目

```bash
npm run build -- --mode linux-production
.\build-linux-production.bat # windows bat启动
```
构建测试环境项目
```bash
npm run build -- --mode docker-test-local
.\build-docker-test-local.bat # windows bat启动
```
#### 清理构建文件
```bash
npm run clean
```
